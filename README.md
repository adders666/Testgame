# Testgame

This repository contains a simple word processor example implemented with Tkinter.

## Requirements

Install dependencies with:

```bash
python3 -m pip install python-docx PyPDF2 striprtf beautifulsoup4 reportlab
```

## Usage

Run the application with:

```bash
python3 word_processor.py
```

The program supports basic text formatting (bold and italic), inserting page breaks, and importing or exporting text, RTF, DOCX, PDF, or HTML files. Imported PDFs and HTML files are converted to plain text before insertion.
